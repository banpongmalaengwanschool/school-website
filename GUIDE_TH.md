## 🔄 วิธีอัปเดตข่าวสาร

### ขั้นตอนที่ 1: เปิดไฟล์ JSON

1. เปิดโฟลเดอร์ `school-website`
2. เข้าไปในโฟลเดอร์ `data`
3. เปิดไฟล์ `news.json` ด้วย Text Editor (Notepad, VS Code, Sublime)

### ขั้นตอนที่ 2: เพิ่มข่าวสารใหม่

ไฟล์ `news.json` มีลักษณะดังนี้:

```json
{
  "news": [
    {
      "id": 1,
      "title": "ยินดีต้อนรับเข้าปีการศึกษา 2568",
      "category": "ประกาศ",
      "date": "2025-02-20",
      "description": "โรงเรียนบ้านโป่งแมลงวัน...",
      "image": "news-images/welcome-2568.jpg",
      "files": []
    }
  ]
}
```

**อธิบายแต่ละส่วน:**

| ส่วน | ความหมาย | ตัวอย่าง |
|------|---------|---------|
| `id` | เลขที่ข่าว (ต้องไม่ซ้ำ) | 1, 2, 3 |
| `title` | ชื่อข่าวสาร | "ประกาศปิดเรียน" |
| `category` | หมวดหมู่ | "ประกาศ", "ปฏิทิน", "สมัครเรียน" |
| `date` | วันที่ (รูปแบบ YYYY-MM-DD) | "2025-02-20" |
| `description` | รายละเอียดข่าว | "ข้อความยาว..." |
| `image` | ที่อยู่รูปภาพ | "news-images/file.jpg" |
| `files` | ไฟล์ดาวโหลด (ถ้ามี) | [ ] หรือ [ {...} ] |

### ขั้นตอนที่ 3: เพิ่มข่าวสารใหม่

ค้นหาบรรทัดสุดท้ายของ `news` และเพิ่มข่าวสารใหม่:

```json
{
  "news": [
    {
      "id": 1,
      "title": "ยินดีต้อนรับเข้าปีการศึกษา 2568",
      ...
    },
    {
      "id": 2,
      "title": "ประกาศปิดเรียนช่วงปิดเทอม",
      "category": "ประกาศ",
      "date": "2025-03-01",
      "description": "โรงเรียนจะปิดเรียนตั้งแต่วันที่ 1 มีนาคม ถึง 15 มีนาคม",
      "image": "news-images/closure.jpg",
      "files": []
    }
  ]
}
```

### ขั้นตอนที่ 4: บันทึกไฟล์

1. กด `Ctrl + S` (Windows) หรือ `Cmd + S` (Mac)
2. ตรวจสอบให้แน่ใจว่าบันทึกเป็น UTF-8 encoding

## 🖼️ วิธีเพิ่มรูปภาพ

### ขั้นตอนที่ 1: เตรียมรูปภาพ

1. **ขนาดแนะนำ:** 1200x600px หรือ 800x400px
2. **รูปแบบ:** JPG, PNG
3. **ขนาดไฟล์:** ไม่เกิน 5 MB

### ขั้นตอนที่ 2: บันทึกรูปภาพ

1. เปิดโฟลเดอร์ `school-website/news-images`
2. คัดลอกรูปภาพลงในโฟลเดอร์นี้
3. ใช้ชื่อไฟล์ที่เข้าใจง่าย เช่น:
   - `news-1.jpg`
   - `event-2025.jpg`
   - `closure-march.jpg`

### ขั้นตอนที่ 3: อัปเดต JSON

แก้ไขไฟล์ `data/news.json` และใส่ URL รูปภาพ:

```json
{
  "id": 2,
  "title": "ประกาศปิดเรียนช่วงปิดเทอม",
  "image": "news-images/closure-march.jpg",  ← ใส่ชื่อไฟล์ที่นี่
  ...
}
```

## 📥 วิธีเพิ่มไฟล์ดาวโหลด

### ขั้นตอนที่ 1: เตรียมไฟล์

1. เตรียมไฟล์ที่ต้องการให้คนดาวโหลด (PDF, Word, Excel, ZIP)
2. ใช้ชื่อไฟล์ที่เข้าใจง่าย

### ขั้นตอนที่ 2: บันทึกไฟล์

1. เปิดโฟลเดอร์ `school-website/downloads`
2. คัดลอกไฟล์ลงในโฟลเดอร์นี้

### ขั้นตอนที่ 3: อัปเดต JSON

**วิธีที่ 1: เพิ่มในข่าวสาร**

แก้ไขไฟล์ `data/news.json`:

```json
{
  "id": 1,
  "title": "ประกาศรับสมัครนักเรียนใหม่",
  "files": [
    {
      "id": 1,
      "name": "แบบฟอร์มสมัครเรียน.docx",
      "url": "downloads/admission-form.docx",
      "size": "1.2 MB",
      "type": "word"
    },
    {
      "id": 2,
      "name": "ประกาศรับสมัครเรียน.pdf",
      "url": "downloads/admission-announcement.pdf",
      "size": "0.8 MB",
      "type": "pdf"
    }
  ]
}
```

**วิธีที่ 2: เพิ่มในหน้าดาวโหลด**

แก้ไขไฟล์ `data/downloads.json`:

```json
{
  "id": 5,
  "category": "เอกสาร",
  "title": "แบบรายงานผลการเรียน",
  "description": "แบบรายงานผลการเรียนสำหรับครูกรอก",
  "file": {
    "name": "report-form.xlsx",
    "url": "downloads/report-form.xlsx",
    "size": "0.5 MB",
    "type": "excel"
  },
  "date": "2025-02-21

}
```

### ประเภทไฟล์ที่รองรับ

| ประเภท | ค่า | ตัวอย่าง |
|--------|-----|---------|
| PDF | `pdf` | `document.pdf` |
| Word | `word` | `form.docx` |
| Excel | `excel` | `data.xlsx` |
| ZIP | `zip` | `files.zip` |

## ❓ แก้ไขปัญหาทั่วไป

### ปัญหา: ข่าวสารไม่แสดงบนเว็บไซต์

**สาเหตุ:** JSON มีข้อผิดพลาด

**วิธีแก้:**
1. เปิดไฟล์ `data/news.json`
2. ตรวจสอบ:
   - วงเล็บปิด `}` ครบหรือไม่
   - เครื่องหมายคอมมา `,` ถูกต้องหรือไม่
   - ข้อความในเครื่องหมายอัญประลัษ `"` ครบหรือไม่
3. ใช้ [JSONLint](https://jsonlint.com/) เพื่อตรวจสอบ

### ปัญหา: รูปภาพไม่แสดง

**สาเหตุ:** ที่อยู่รูปภาพผิด

**วิธีแก้:**
1. ตรวจสอบชื่อไฟล์ใน JSON
2. ตรวจสอบว่าไฟล์อยู่ในโฟลเดอร์ `news-images/`
3. ตรวจสอบสะกดชื่อไฟล์ให้ถูกต้อง (ตัวพิมพ์เล็ก/ใหญ่)

### ปัญหา: ไฟล์ดาวโหลดไม่ทำงาน

**สาเหตุ:** ไฟล์ไม่อยู่ในโฟลเดอร์ `downloads/`

**วิธีแก้:**
1. ตรวจสอบว่าไฟล์อยู่ในโฟลเดอร์ `downloads/`
2. ตรวจสอบชื่อไฟล์ใน JSON ให้ตรงกับชื่อไฟล์จริง

### ปัญหา: เว็บไซต์ไม่อัปเดต

**สาเหตุ:** ยังไม่ push ไปยัง GitHub

**วิธีแก้:**
```bash
git add .
git commit -m "Update"
git push
```

## 💡 เคล็ดลับการใช้งาน

1. **ใช้ชื่อไฟล์ที่เข้าใจง่าย** - ใช้ `-` แทนช่องว่าง
   - ✅ `admission-form.docx`
   - ❌ `admission form.docx`

2. **ตรวจสอบ JSON ก่อนบันทึก** - ใช้ [JSONLint](https://jsonlint.com/)

3. **ใช้วันที่ในรูปแบบ YYYY-MM-DD** - เพื่อให้เรียงลำดับได้ถูกต้อง
   - ✅ `2025-02-20`
   - ❌ `20-02-2025`

4. **ทดสอบเว็บไซต์ในเบราว์เซอร์** - ก่อน push ไปยัง GitHub

5. **ใช้ UTF-8 encoding** - เพื่อให้ภาษาไทยแสดงถูกต้อง
